安装说明

各平台安装

Windows

下载解压后，进入程序所在目录（设置环境变量后可在任意目录），Ctrl + 右键「在此处打开 PowerShell」，输入：

./passwordz.exe
macOS

下载解压后，命令行进入程序所在目录（放入 $PATH 可在任意位置），输入 passwordz。若提示「无法打开，无法验证开发者」：点取消 → 系统偏好设置 → 安全性与隐私 → 通用 → 点击「仍要打开」。

Linux

下载解压后，命令行进入程序所在目录（放入 $PATH 可在任意位置），输入 passwordz。

Android

该程序为命令行程序，需使用命令行或 Termux。命令行下载解压后，进入程序所在目录（放入 $PATH 可在任意位置），输入 passwordz。
