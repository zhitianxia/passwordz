Windows Installation instructions
Download and decompression, go to the directory where the program is located (if you set the PATH variables, you can arbitrary the directory), ctrl+right-click to open the PowerShell window here, and then enter the following command to use PasswordZ: 
            
./passwordz.exe
MacOS Installation instructions
Download and decompression,Use the command line to go to the directory where the program is located (put passwordz in the $PATH directory to get anywhere on the command line), and type the passwordz command to use PasswordZ.If it pops up: "passwordz" cannot be opened because the developer cannot be verified. Click Cancel, then go to System Preferences -> Security & Privacy -> General -> in the Running apps downloaded from: The use of "passwordz" has been blocked because it comes from an unidentified developer area. Click Still Running and that's it.
