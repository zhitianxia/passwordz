Online version — no download, use anywhere

Extension

Browser extension

If you're using Microsoft Edge, get the PasswordZ extension directly from the official Edge marketplace.

Desktop

Desktop version

Graphical UI — no command line needed, ~4 MB; identical algorithm to the online and CLI versions.

OSDownloadMD5

Windows: extract and double-click passwordz-desktop.exe (a SmartScreen prompt may appear on first run — choose "Run anyway"); Linux: extract then run chmod +x passwordz-desktop && ./passwordz-desktop; the macOS build must be produced on a Mac and will arrive in a later release.

CLI

Offline command-line version

Single-file command-line program, ~700 KB; identical algorithm to the online and desktop versions.

OSDownloadMD5

Note: v1.1.0 fixes a punctuation-fallback defect, so it differs from v1.0.0 in about 2.55% of all-charset (digit + lower + upper + punctuation) scenarios — please verify the affected passwords before rolling back.

Install

Installation by platform

Windows

Download and extract, then enter the program's directory (add it to PATH to run from anywhere). Ctrl + right-click → "Open PowerShell here", then run:

./passwordz.exe
macOS

Download and extract, then cd into the program's directory (add it to PATH to run from anywhere) and run passwordz. If macOS says it "can't be opened because it is from an unidentified developer": cancel, then go to System Settings → Privacy & Security → click "Open Anyway".

Linux

Download and extract, then cd into the program's directory (add it to PATH to run from anywhere) and run passwordz.

Android

This is a command-line program and needs a terminal or Termux. Download and extract, enter the program's directory (add it to PATH to run from anywhere) and run passwordz.
